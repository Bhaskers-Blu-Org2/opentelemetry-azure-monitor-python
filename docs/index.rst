
OpenTelemetry Azure Monitor Python
==================================

The Python `OpenTelemetry <https://opentelemetry.io/>`_ Azure Monitor library.

.. image:: https://img.shields.io/gitter/room/Microsoft/azure-monitor-python
   :target: https://gitter.im/Microsoft/azure-monitor-python
   :alt: Gitter Chat


**Please note** that this library is currently in beta, and shouldn't be
used in production environments.

Installation
------------

The package is available on PyPI, and can installed via pip:

.. code-block:: sh

    pip install opentelemetry-azure-monitor-exporter
